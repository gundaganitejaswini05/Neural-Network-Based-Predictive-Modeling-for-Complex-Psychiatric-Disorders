Neural Network-Based Predictive Modeling for Complex Psychiatric Disorder Identification from EEG Features

Welcome to the EEG Psychiatric Disorder Classification System

This application uses advanced machine learning techniques to classify psychiatric disorders based on EEG (Electroencephalography) features.

System Features:
  • Multiple Classification Models: Random Forest, Support Vector Machine, Logistic Regression, Decision Tree, and Hybrid Graph Neural Network (HGNN)
  • Two-Level Classification:
    • Main Disorder Classification
    • Specific Disorder Classification
  • Comprehensive Performance Metrics: Accuracy, Precision, Recall, F1-Score, Confusion Matrix, and Classification Report
  • Model Persistence: Trained models are saved and can be reused without retraining
